Папка images предназначена для скриншотов и любых изображений, которые вы хотите использовать в галерее сайта.

Пример:
- 1.png
- 2.png
- 3.png

Положите сюда свои файлы (jpg, png и т.д.) и укажите их имена в массиве galleryItems в app/page.tsx.

Путь для использования: /images/1.png

Если вы добавили новые изображения, не забудьте перезапустить сервер разработки.
